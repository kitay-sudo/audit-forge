Yata — это один самодостаточный .md-файл, который превращает любого агента — Claude Code, Cursor, Codex — в дотошного аудитора безопасности. Понимает любой стек, проходит 17 фаз и закрывает дыры готовыми diff-патчами.

Инструмент, которому доверяют

Свод признанных стандартов индустрии в одном файле — OWASP, CWE, NIST, SLSA. Агент работает как senior AppSec-инженер: каждая находка с реальным путём эксплуатации, без «воды».

Фиксы как diff, а не советы

Каждая уязвимость закрывается минимальным unified-diff патчем под git apply — идиоматично коду, без поломки логики.

yata/fixes/
├── CRIT-01-idor.diff git apply ✔
├── HIGH-01-sqli.diff
└── HIGH-02-ssrf.diff

Результат — одна папка yata/

Самодостаточно и готово к ревью. Остальной репозиторий агент не трогает.

yata/
├── README.md вердикт + severity
├── SECURITY_AUDIT_REPORT.md
├── findings/ · fixes/
└── appendix/ сырой вывод SCA

Готов к проду без страха

Финальный production-readiness gate даёт чёткий вердикт go / no-go.

  • ✓AI / LLM — OWASP LLM Top 10 (2025): prompt injection, excessive agency, cost-DoS
  • ✓Supply chain — CVE, SBOM, SLSA, подпись артефактов
  • ✓Rate limit и нагрузка — чтобы не упасть и не утечь

Работает с любым стеком.

Агент сам определяет язык и фреймворк по файлам-маркерам — и подбирает правильные проверки и инструменты.

⚠️ Только для авторизованного аудита

Yata — для защитной проверки кода, который вам разрешено аудитировать: свой проект, проект работодателя, клиент с договором, CTF или исследование с разрешением. По умолчанию статический/локальный анализ; динамические тесты — только против сред, которыми вы владеете. Секреты редактируются, а не выводятся. Это навигатор для агента, а не гарантия — результат служит экспертным входом для решения человека.
